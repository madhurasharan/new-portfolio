# Madhura Portfolio

A modern dark developer portfolio built with React, Vite, Tailwind CSS, Framer Motion, and React Icons.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Customize

- Replace the social links in `src/components/Contact.jsx` and `src/components/Footer.jsx`
- Replace the GitHub username in `src/components/GitHubStats.jsx`
- Replace `public/Madhura-SN-Resume.txt` with your actual resume PDF if needed
- Update project/demo links in `src/components/Projects.jsx`
